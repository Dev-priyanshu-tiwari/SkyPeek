# ☁️ SkyPeek — Weather Dashboard

A colorful, interactive weather dashboard built with **HTML, CSS & Vanilla JavaScript**, using the **OpenWeatherMap API**.

Test NOW - https://dev-priyanshu-tiwari.github.io/SkyPeek/

---

## 🌟 Features

- 🔍 **City Search** with quick-access buttons for popular cities
- 🌡️ **Real-time Weather** — temperature, humidity, wind, visibility, pressure
- 📅 **5-Day Forecast** with weather icons
- 🔄 **°C / °F Toggle** — switch units instantly
- 🌅 **Sunrise & Sunset times**
- ☁️ UV Index, Cloud Cover, Coordinates
- 🎨 Animated gradient blobs + playful card UI
- 📱 Fully responsive on mobile and desktop
- 🧪 **Wheather API is Used**
